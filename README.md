# O Sonho de Jorge - O endless runner de carnaval! 

 
## Sobre o Jogo
O sonho de Jorge é um endless runner de corrida 100% brasileiro que utiliza Jorge como personagem principal, distribuindo preservativos durante o Carnaval.
Ele usa a mecânica endless runner, sendo hipercasual com uma interação simples.


## Informações
Você pode ter mais informações acessando - [o mini site](https://jorgesdream.carrd.co/) do jogo.

## Créditos
Esse jogo foi desenvolvido como uma atividade da Apple Developer Academy Curitiba, pelas alunas Barbra Eliza a Djenifer Pereira.
